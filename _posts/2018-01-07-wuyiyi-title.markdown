# 怎么用 MarkDown 写博客？

1. 学习 markdown 语法，参考我在知乎写的[攻略](https://zhuanlan.zhihu.com/p/31784246)。

2. 安装md编辑器--**Yu Writer**,并对「.md」和「.markdown」格式关联

3. 开动。

4. 发布。简书支持 MD。也可以在github上搭建博客平台。
